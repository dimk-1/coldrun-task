# coldrun-task

## Project Setup

```sh
pnpm install
```

## Environmental variables

`API_HOST` - is used to interact with the remote server

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
